# SkypeClone
Building SkypeClone using Material-UI, React.js, Express.js, Node.js, MongoDB &amp; WebRTC

## Installation 

```
git clone https://github.com/aemal/SkypeClone.git
cd skypeClone
npm install
```
## Running the app
### Run client side
```
npm start
```
### Run server side
```
node app.js
```

### Run tests
open test console `./node_modules/.bin/cypress open`

run `dashboard_spec.js`
